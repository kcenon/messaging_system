Detailed Description

Specialization of std::formatter for thread_module::thread_conditions.

Allows thread_conditions enum values to be formatted as strings using the C++20 <format> library (when USE_STD_FORMAT is defined).

Example

auto cond = thread_module::thread_conditions::Working;
std::string output = std::format("Thread state is {}", cond); // "Thread state is working"
thread_module::thread_conditions::Working
@ Working
Thread currently processing a task.
