Detailed Description

template<typename T>
class kcenon::logger::lightweight_di_container< T >

Lightweight implementation of DI container.

Template Parameters
TThe type of objects this container manages

This container provides thread-safe registration and resolution of components using factories and singletons, with no external dependencies.
