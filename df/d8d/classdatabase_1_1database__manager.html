Detailed Description

Manages database connections and operations in a singleton pattern.

The database_manager class provides a high-level interface for controlling database connections and executing queries. It wraps a database_base instance and exposes methods such as connect, disconnect, create_query, insert_query, etc. This class also follows the singleton design pattern, allowing only one instance of the manager to exist at a time.
