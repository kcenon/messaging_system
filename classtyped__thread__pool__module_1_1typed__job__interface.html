Detailed Description

template<detail::JobType job_type>
class typed_thread_pool_module::typed_job_interface< job_type >

Base interface for all typed jobs.

This interface defines the contract that all typed jobs must follow. It provides type information and execution capabilities while maintaining type safety through templates.
