Detailed Description

Specialization of std::formatter for thread_pool_module::thread_worker.

Allows thread_worker objects to be formatted as strings using the C++20 <format> library (when USE_STD_FORMAT is defined).

Example

auto worker = std::make_unique<thread_pool_module::thread_worker>();
std::string output = std::format("Worker status: {}", *worker);
