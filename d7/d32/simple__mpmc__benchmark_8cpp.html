Detailed Description

Google Benchmark tests for MPMC queue implementations.

This file benchmarks different queue implementations:

  • Mutex-based queue (job_queue)
  • Lock-free MPMC queue (lockfree_job_queue)
  • Adaptive queue (adaptive_job_queue)

Tests measure throughput, latency, and scalability under various contention scenarios following Google Benchmark best practices.
