Detailed Description

template<typename job_type>
struct fmt::formatter< typed_thread_pool_module::typed_job_queue_t< job_type > >

Specialization of fmt::formatter for typed_job_queue_t<job_type> when using the {fmt} library.

This allows a typed_job_queue_t<job_type> to be formatted using the {fmt} library, by invoking the to_string() method on the instance.

Template Parameters
job_typeThe type representing priority levels.
