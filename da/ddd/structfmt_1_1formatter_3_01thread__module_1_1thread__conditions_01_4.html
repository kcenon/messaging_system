Detailed Description

Specialization of fmt::formatter for thread_module::thread_conditions.

Allows thread_conditions enum values to be formatted as strings using the {fmt} library.

Example

auto cond = thread_module::thread_conditions::Working;
std::string output = fmt::format("Thread state is {}", cond); // "Thread state is working"
thread_module::thread_conditions::Working
