Detailed Description

Specialization of fmt::formatter for kcenon::thread::thread_conditions.

Allows thread_conditions enum values to be formatted as strings using the {fmt} library.

Example

auto cond = kcenon::thread::thread_conditions::Working;
std::string output = fmt::format("Thread state is {}", cond); // "Thread state is working"
kcenon::thread::thread_conditions::Working
@ Working
Thread currently processing a task.
