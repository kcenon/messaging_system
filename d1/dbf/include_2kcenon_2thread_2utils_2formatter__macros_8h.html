Detailed Description

Provides macros for generating formatter specializations.

This file contains macros that generate formatter specializations for custom types to work with both std::format (C++20) and fmt library. The macros eliminate code duplication by generating the necessary boilerplate code for formatter specializations.

Usage example:

// In your header file after class definition:
DECLARE_FORMATTER(my_namespace::my_class)
DECLARE_FORMATTER
#define DECLARE_FORMATTER(CLASS_NAME)
