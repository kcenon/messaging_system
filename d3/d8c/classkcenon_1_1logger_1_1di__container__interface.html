Detailed Description

template<typename T>
class kcenon::logger::di_container_interface< T >

Abstract interface for dependency injection container.

Template Parameters
TThe type of objects this container manages

This interface defines the contract for DI containers, allowing different implementations without coupling to specific DI frameworks.
