Detailed Description

A server class that manages incoming TCP connections, creating messaging_session instances for each accepted socket.

Key Responsibilities

  • Maintains an asio::io_context and tcp::acceptor to listen on a specified port.
  • For each incoming connection, instantiates a messaging_session to handle the communication logic (compression, encryption, message parsing, etc.).
  • Allows external control via start_server(), stop_server(), and wait_for_stop().

Thread Model

  • A single background thread calls io_context.run() to process I/O events.
  • Each accepted connection runs asynchronously; thus multiple sessions can be active concurrently without blocking each other.

Usage Example

// 1. Create a messaging_server:
auto server = std::make_shared<network::messaging_server>("ServerID");
// 2. Start listening on a port:
server->start_server(5555);
// 3. Wait for stop or run other tasks:
// ...
// 4. Eventually stop the server:
server->stop_server();
// 5. (optional) If a separate thread is calling wait_for_stop():
server->wait_for_stop();

Begins listening on the specified TCP port, creates a background thread to run I/O operations, and starts accepting connections.

Parameters
portThe TCP port to bind and listen on (e.g., 5555).

Behavior

  • If the server is already running (is_running_ is true), this call does nothing.
  • Otherwise, an io_context and acceptor are created, do_accept() is invoked, and a new thread is spawned to run io_context->run().

◆ stop_server()

auto network_module::messaging_server::stop_server ( ) -> void

Stops the server, closing the acceptor and all active sessions, then stops the io_context and joins the internal thread.

Steps:

  1. Set is_running_ to false.
  2. Close the acceptor if open.
  3. Iterate through all active sessions, calling stop_session().
  4. io_context->stop() to halt asynchronous operations.
  5. Join the background thread if it's joinable.
  6. Fulfill the stop_promise_ so that wait_for_stop() can return.
Note
If the server is not running, this function does nothing.
