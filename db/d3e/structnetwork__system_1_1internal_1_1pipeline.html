Detailed Description

Holds function objects for compressing, decompressing, encrypting, and decrypting data.

Typically used by messaging_session or messaging_client to transform data buffers before sending or after receiving. The default stubs do no actual transformation.

Example

// Create a default pipeline
pipeline p = make_default_pipeline();
// Use p.compress(...) on data
auto compressed = p.compress(original_data);
// ...
