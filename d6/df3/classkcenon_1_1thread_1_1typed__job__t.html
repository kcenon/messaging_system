Detailed Description

template<typename job_type>
class kcenon::thread::typed_job_t< job_type >

Represents a job that carries a specific priority level.

This class extends the base job interface to include a priority value that is used by a priority-based scheduling system. The class also keeps a weak reference to a typed_job_queue_t, which manages this job. By using a weak pointer, the queue avoids circular references that could prevent proper resource cleanup.

Template Parameters
job_typeThe data type used to represent the priority level. Typically, an enum or an integral type.
